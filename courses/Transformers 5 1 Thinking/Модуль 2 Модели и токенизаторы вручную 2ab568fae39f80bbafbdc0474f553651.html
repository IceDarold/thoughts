–£—Ä–æ–∫ 2.1. –†–∞–∑–±–∏—Ä–∞–µ–º pipeline –Ω–∞ —á–∞—Å—Ç–∏: AutoTokenizer + AutoModelForSequenceClassification

–ü–æ–∫–∞–∑–∞—Ç—å, –∫–∞–∫ —Å–¥–µ–ª–∞—Ç—å —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ pipeline("sentiment-analysis"), –Ω–æ —Ä—É–∫–∞–º–∏:

2.1.1. –ö–µ–π—Å

–ú—ã —Ö–æ—Ç–∏–º –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, —á—Ç–æ –∏–º–µ–Ω–Ω–æ –¥–µ–ª–∞–µ—Ç –Ω–∞—à sentiment-–∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä —Ç–∏–∫–µ—Ç–æ–≤.

–ò –Ω–∞—É—á–∏—Ç—å—Å—è, –Ω–∞–ø—Ä–∏–º–µ—Ä, –º–µ–Ω—è—Ç—å –ø–æ—Ä–æ–≥ —É–≤–µ—Ä–µ–Ω–Ω–æ—Å—Ç–∏, —Å–∞–º–∏ —Å—á–∏—Ç–∞—Ç—å softmax –∏ —Ç.–¥.

2.1.2. –¢–µ–æ—Ä–∏—è: —á—Ç–æ —Ç–∞–∫–æ–µ —Ç–æ–∫–µ–Ω–∏–∑–∞—Ç–æ—Ä –∏ –º–æ–¥–µ–ª—å

–û–±—ä—è—Å–Ω—è–µ–º –∫–æ—Ä–æ—Ç–∫–æ:

  • Tokenizer:
    • –ø—Ä–µ–≤—Ä–∞—â–∞–µ—Ç —Ç–µ–∫—Å—Ç –≤ input_ids (–Ω–æ–º–µ—Ä–∞ —Ç–æ–∫–µ–Ω–æ–≤),
    • –¥–æ–±–∞–≤–ª—è–µ—Ç —Å–ø–µ—Ü. —Ç–æ–∫–µ–Ω—ã ([CLS], [SEP]),
    • —Å–æ–∑–¥–∞—ë—Ç attention_mask (–≥–¥–µ –ø–∞–¥–¥–∏–Ω–≥, –≥–¥–µ –Ω–µ—Ç),
    • –º–æ–∂–µ—Ç –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å —Ç–µ–Ω–∑–æ—Ä—ã —Å—Ä–∞–∑—É –≤ torch.
  • ModelForSequenceClassification:
    • –±–µ—Ä—ë—Ç input_ids, attention_mask,
    • –¥–µ–ª–∞–µ—Ç encoder,
    • —Å–≤–µ—Ä—Ö—É ‚Äî –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ü–∏–æ–Ω–Ω–∞—è –≥–æ–ª–æ–≤–∞,
    • –Ω–∞ –≤—ã—Ö–æ–¥–µ:
      • logits —Ä–∞–∑–º–µ—Ä–æ–º [batch_size, num_labels],

2.1.3. –ö–æ–¥: —Ä—É—á–Ω–æ–π sentiment (–æ–¥–∏–Ω —Ç–µ–∫—Å—Ç)

import torch
from transformers import AutoTokenizer, AutoModelForSequenceClassification

model_id = "distilbert-base-uncased-finetuned-sst-2-english"  # –ø—Ä–∏–º–µ—Ä

tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForSequenceClassification.from_pretrained(model_id)

text = "I absolutely love this app, it's super useful!"

# –¢–æ–∫–µ–Ω–∏–∑–∞—Ü–∏—è
enc = tokenizer(
    text,
    return_tensors="pt",   # –≤–µ—Ä–Ω—ë—Ç PyTorch —Ç–µ–Ω–∑–æ—Ä—ã
    padding=True,
    truncation=True,
)

print(enc.keys())
# dict_keys(['input_ids', 'attention_mask'])

print(enc["input_ids"].shape)      # torch.Size([1, seq_len])
print(enc["attention_mask"].shape) # torch.Size([1, seq_len])

# –ü—Ä–æ–≥–æ–Ω —á–µ—Ä–µ–∑ –º–æ–¥–µ–ª—å
with torch.no_grad():  # –∏–Ω—Ñ–µ—Ä–µ–Ω—Å –±–µ–∑ –≥—Ä–∞–¥–∏–µ–Ω—Ç–æ–≤
    outputs = model(**enc)

print(type(outputs))
print(outputs.keys())
# dict_keys(['logits'])

logits = outputs.logits  # shape: [1, num_labels]
print(logits.shape)

probs = torch.softmax(logits, dim=-1)
print("Probs:", probs)

pred_id = torch.argmax(probs, dim=-1).item()
id2label = model.config.id2label
print("Predicted label:", id2label[pred_id])

–ß—Ç–æ –æ–±—ä—è—Å–Ω—è–µ–º:

  • return_tensors="pt" ‚Üí –ø–æ–ª—É—á–∞–µ–º PyTorch-—Ç–µ–Ω–∑–æ—Ä—ã, –Ω–µ —Å–ø–∏—Å–∫–∏.
  • model(**enc) ‚Äî —É–¥–æ–±–Ω—ã–π –≤—ã–∑–æ–≤ —Å —Ä–∞—Å–ø–∞–∫–æ–≤–∫–æ–π —Å–ª–æ–≤–∞—Ä—è.
  • logits ‚Äî ¬´—Å—ã—Ä—ã–µ¬ª –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—è; softmax –¥–µ–ª–∞–µ–º —Å–∞–º–∏.
  • model.config.id2label –∏ label2id ‚Äî —Å–≤—è–∑—å –∏–Ω–¥–µ–∫—Å–∞ –∏ —Ç–µ–∫—Å—Ç–æ–≤–æ–≥–æ –∫–ª–∞—Å—Å–∞.

2.1.4. –ë–∞—Ç—á —Ç–∏–∫–µ—Ç–æ–≤

–ü–µ—Ä–µ–¥–µ–ª–∞–µ–º –Ω–∞ –±–∞—Ç—á (–ø–æ–¥ –Ω–∞—à –∫–µ–π—Å —Ç–∏–∫–µ—Ç–æ–≤ –ø–æ–¥–¥–µ—Ä–∂–∫–∏):

tickets = [
    "The app crashes when I try to upload a photo.",
    "Can you add dark mode, please?",
    "I was charged twice this month.",
    "Great app, just wanted to say thanks!"
]

enc = tokenizer(
    tickets,
    return_tensors="pt",
    padding=True,
    truncation=True,
)

with torch.no_grad():
    outputs = model(**enc)

logits = outputs.logits               # [batch, num_labels]
probs = torch.softmax(logits, dim=-1) # [batch, num_labels]

for text, pred_id, p in zip(tickets, pred_ids, probs):

    confidence = p[pred_id].item()
    print(f"TEXT: {text}\nPRED: {label} ({confidence:.3f})\n")

  1. –î–æ–±–∞–≤–∏—Ç—å –æ–ø—Ü–∏—é: –≤—ã–≤–æ–¥–∏—Ç—å –≤—Å–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –ø–æ –∫–ª–∞—Å—Å–∞–º (—Ç–∞–±–ª–∏—á–∫–æ–π).

–£—Ä–æ–∫ 2.2. –ò—Å–ø–æ–ª—å–∑—É–µ–º —Ç—Ä–∞–Ω—Å—Ñ–æ—Ä–º–µ—Ä –∫–∞–∫ feature extractor

–¶–µ–ª—å:

–ü–æ–∫–∞–∑–∞—Ç—å, –∫–∞–∫ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å AutoModel (–±–µ–∑ –≥–æ–ª–æ–≤—ã) –¥–ª—è:

  • –∏–∑–≤–ª–µ—á–µ–Ω–∏—è —ç–º–±–µ–¥–¥–∏–Ω–≥–æ–≤ —Ç–∏–∫–µ—Ç–æ–≤,
  • –æ–±—É—á–µ–Ω–∏—è —Å–≤–æ–µ–≥–æ –ø—Ä–æ—Å—Ç–æ–≥–æ –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, LogisticRegression –≤ sklearn).

–≠—Ç–æ —É–∂–µ –º–æ—Å—Ç–∏–∫ –∫ –∫–∞—Å—Ç–æ–º–Ω—ã–º –ø–∞–π–ø–ª–∞–π–Ω–∞–º.

2.2.1. –ö–µ–π—Å

–ù–∞–º –Ω—É–∂–Ω–æ –∫–ª–∞—Å—Å–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞—Ç—å —Ç–∏–∫–µ—Ç—ã –ø–æ —Å–≤–æ–∏–º –∫–ª–∞—Å—Å–∞–º

–Ω–æ –≥–æ—Ç–æ–≤–æ–π –º–æ–¥–µ–ª–∏ –ø–æ–¥ —ç—Ç–æ –Ω–µ—Ç.

  • –Ω–∞—Ç—Ä–∞–≤–∏—Ç—å —Å–≤–µ—Ä—Ö—É –ø—Ä–æ—Å—Ç–æ–π –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä.

2.2.2. –¢–µ–æ—Ä–∏—è: AutoModel vs AutoModelForSequenceClassification

  • AutoModel:
    • –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ç–æ–ª—å–∫–æ —Å–∫—Ä—ã—Ç—ã–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è (hidden states),
    • –Ω–∏–∫–∞–∫–∏—Ö –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ü–∏–æ–Ω–Ω—ã—Ö –≥–æ–ª–æ–≤.
  • AutoModelForSequenceClassification:
    • –¥–æ–±–∞–≤–ª—è–µ—Ç —Å–≤–µ—Ä—Ö—É Linear–≥–æ–ª–æ–≤—É,
    • –æ–±—É—á–∞–ª—Å—è –ø–æ–¥ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—É—é –∑–∞–¥–∞—á—É.

–ú—ã —Å–¥–µ–ª–∞–µ–º —Ç–∞–∫:

  1. AutoModel ‚Üí [CLS]–≤–µ–∫—Ç–æ—Ä / –ø—É–ª–ª–∏–Ω–≥ ‚Üí —ç–º–±–µ–¥–¥–∏–Ω–≥ —Ç–µ–∫—Å—Ç–∞.
  1. –ü—Ä–µ–¥–∞—ë–º —ç—Ç–∏ —ç–º–±–µ–¥–¥–∏–Ω–≥–∏ –≤ sklearn–∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä.

2.2.3. –ö–æ–¥: –≤—ã—Ç–∞—Å–∫–∏–≤–∞–µ–º —ç–º–±–µ–¥–¥–∏–Ω–≥–∏ [CLS]

import torch
from transformers import AutoTokenizer, AutoModel

model_id = "distilbert-base-uncased"  # –±–∞–∑–æ–≤–∞—è –º–æ–¥–µ–ª—å –±–µ–∑ –≥–æ–ª–æ–≤—ã

tokenizer = AutoTokenizer.from_pretrained(model_id)
base_model = AutoModel.from_pretrained(model_id)

texts = [
    "The app crashes when I try to upload a photo.",
    "Can you add dark mode, please?",
    "I was charged twice this month.",
    "Great app, just wanted to say thanks!"
]

enc = tokenizer(
    texts,
    padding=True,
    truncation=True,
    return_tensors="pt",
)

with torch.no_grad():
    outputs = base_model(**enc)

last_hidden_state = outputs.last_hidden_state  # [batch, seq_len, hidden_size]
print(last_hidden_state.shape)

–ë–µ—Ä—ë–º —ç–º–±–µ–¥–¥–∏–Ω–≥ CLS (–ø–µ—Ä–≤—ã–π —Ç–æ–∫–µ–Ω):

print(cls_embeddings.shape)

–¢–µ–ø–µ—Ä—å —ç—Ç–æ –æ–±—ã—á–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ –ø—Ä–∏–∑–Ω–∞–∫–æ–≤.

2.2.4. –£—á–∏–º –ø—Ä–æ—Å—Ç–æ–π –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä –ø–æ–≤–µ—Ä—Ö —ç–º–±–µ–¥–¥–∏–Ω–≥–æ–≤

–ü—Ä–∏–º–µ—Ä —Å –∏–≥—Ä—É—à–µ—á–Ω—ã–º–∏ –¥–∞–Ω–Ω—ã–º–∏ (–≤ —Ä–µ–∞–ª—å–Ω–æ–π –∂–∏–∑–Ω–∏ ‚Äî –Ω—É–∂–Ω–æ –±–æ–ª—å—à–µ):

from sklearn.linear_model import LogisticRegression
import numpy as np

# –î–æ–ø—É—Å—Ç–∏–º, —É –Ω–∞—Å –µ—Å—Ç—å —Ä–∞–∑–º–µ—Ç–∫–∞ –∫–ª–∞—Å—Å–æ–≤ –¥–ª—è —ç—Ç–∏—Ö 4 —Ç–∏–∫–µ—Ç–æ–≤
# 0 = bug, 1 = feature, 2 = billing, 3 = other
labels = np.array([0, 1, 2, 3])

X = cls_embeddings.numpy()

clf.fit(X, labels)

# –ü—Ä–æ–≤–µ—Ä–∏–º –Ω–∞ –Ω–æ–≤—ã—Ö —Ç–∏–∫–µ—Ç–∞—Ö
new_tickets = [
    "The app crashes on launch.",
    "Please add the option to change language.",
    "I don't understand this charge on my credit card.",
]

enc_new = tokenizer(
    new_tickets,
    padding=True,
    truncation=True,
    return_tensors="pt",
)

with torch.no_grad():
    outputs_new = base_model(**enc_new)

cls_new = outputs_new.last_hidden_state[:, 0, :].numpy()
preds = clf.predict(cls_new)

id2label_custom = {0: "bug", 1: "feature", 2: "billing", 3: "other"}

for text, pred in zip(new_tickets, preds):
    print(f"TEXT: {text}\nCLASS: {id2label_custom[pred]}\n")

2.2.5. –î–æ–º–∞—à–∫–∞ 2.2

  1. –í—ã–±—Ä–∞—Ç—å –ª—é–±—É—é –±–∞–∑–æ–≤—É—é –º–æ–¥–µ–ª—å (bert-base-*, distilbert-*, rubert-* –∏ —Ç.–ø.).
  1. –°–æ–±—Ä–∞—Ç—å –Ω–µ–±–æ–ª—å—à–æ–π –Ω–∞–±–æ—Ä —Ç–µ–∫—Å—Ç–æ–≤ (10‚Äì30) —Å —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–º–∏ –∫–ª–∞—Å—Å–∞–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ç–µ–º–∞—Ç–∏–∫–∞: —Ç–µ—Ö–ø–æ–¥–¥–µ—Ä–∂–∫–∞, —Ç–µ–º—ã –Ω–æ–≤–æ—Å—Ç–µ–π, —Ç–∏–ø—ã –æ—Ç–∑—ã–≤–æ–≤).
  1. –°–¥–µ–ª–∞—Ç—å:
    • –∏–∑–≤–ª–µ—á–µ–Ω–∏–µ [CLS]-—ç–º–±–µ–¥–¥–∏–Ω–≥–æ–≤ —á–µ—Ä–µ–∑ AutoModel,
    • –æ–±—É—á–µ–Ω–∏–µ –ø—Ä–æ—Å—Ç–æ–≥–æ –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä–∞ (LogReg / RandomForest),
    • —Ç–µ—Å—Ç –Ω–∞ –Ω–æ–≤—ã—Ö –ø—Ä–∏–º–µ—Ä–∞—Ö.

–£—Ä–æ–∫ 2.3. –ö–æ–Ω—Ñ–∏–≥–∏, label2id/id2label –∏ –∫–∞—Å—Ç–æ–º–Ω—ã–µ –∫–ª–∞—Å—Å—ã

–¶–µ–ª—å:
