2.1. Accuracy

Accuracy ‚Äî –¥–æ–ª—è –ø—Ä–∞–≤–∏–ª—å–Ω–æ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω—ã—Ö –ø—Ä–∏–º–µ—Ä–æ–≤:

[

\text{accuracy} = \frac{\text{–∫–æ–ª-–≤–æ –≤–µ—Ä–Ω–æ –∫–ª–∞—Å—Å–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –æ–±—ä–µ–∫—Ç–æ–≤}}{\text{–æ–±—â–µ–µ –∫–æ–ª-–≤–æ –æ–±—ä–µ–∫—Ç–æ–≤}}

]

–ü—Ä–∏–º–µ—Ä:

–ò–∑ 100 —Ç–∏–∫–µ—Ç–æ–≤ –º–æ–¥–µ–ª—å 90 –ø—Ä–µ–¥—Å–∫–∞–∑–∞–ª–∞ –ø—Ä–∞–≤–∏–ª—å–Ω–æ ‚Üí accuracy = 0.9.

–ü—Ä–æ–±–ª–µ–º–∞: –ø—Ä–∏ –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∫–ª–∞—Å—Å–∞—Ö accuracy –º–æ–∂–µ—Ç –≤—Ä–∞—Ç—å.

–ü—Ä–∏–º–µ—Ä:

  • 95% —Ç–∏–∫–µ—Ç–æ–≤ ‚Äî –∫–ª–∞—Å—Å other,
  • 5% ‚Äî billing.

    –ú–æ–¥–µ–ª—å –≤—Å–µ–≥–¥–∞ –≥–æ–≤–æ—Ä–∏—Ç other:

  • accuracy = 95%,
  • –Ω–æ –ø–æ —Ñ–∞–∫—Ç—É –æ–Ω–∞ –Ω–∏–∫–æ–≥–¥–∞ –Ω–µ –≤–∏–¥–∏—Ç billing.

2.2. Precision, Recall –∏ F1 (–Ω–∞ –ø–∞–ª—å—Ü–∞—Ö)

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –æ–¥–∏–Ω –∫–ª–∞—Å—Å, –Ω–∞–ø—Ä–∏–º–µ—Ä billing.

  • Precision (—Ç–æ—á–Ω–æ—Å—Ç—å):
    • –∏–∑ –≤—Å–µ–≥–æ, —á—Ç–æ –º–æ–¥–µ–ª—å –Ω–∞–∑–≤–∞–ª–∞ billing, –∫–∞–∫–∞—è –¥–æ–ª—è –¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ billing?
    • ‚Äú–Ω–∞—Å–∫–æ–ª—å–∫–æ –≥—Ä—è–∑–Ω—ã–π —É –Ω–∞—Å –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω—ã–π –∫–ª–∞—Å—Å‚Äù (–º–Ω–æ–≥–æ –ª–∏ ‚Äú–º—É—Å–æ—Ä–∞‚Äù).
  • Recall (–ø–æ–ª–Ω–æ—Ç–∞):
    • –∏–∑ –≤—Å–µ—Ö –Ω–∞—Å—Ç–æ—è—â–∏—Ö billing –≤ –¥–∞–Ω–Ω—ã—Ö –∫–∞–∫—É—é –¥–æ–ª—é –º–æ–¥–µ–ª—å –ø–æ–π–º–∞–ª–∞?
    • ‚Äú—Å–∫–æ–ª—å–∫–æ –æ–±—ä–µ–∫—Ç–æ–≤ –º—ã –ø—Ä–æ–ø—É—Å—Ç–∏–ª–∏‚Äù.

–ò–Ω–æ–≥–¥–∞ –≤–∞–∂–Ω–µ–µ:

  • –Ω–µ –ø—Ä–æ–ø—É—Å–∫–∞—Ç—å billing (–≤—ã—Å–æ–∫–∏–π recall),
  • –∏–Ω–æ–≥–¥–∞ ‚Äî –Ω–µ ‚Äú–∑–∞–ª–∏–≤–∞—Ç—å‚Äù –≤—Å—ë –ø–æ–¥—Ä—è–¥ –≤ billing (–≤—ã—Å–æ–∫–∏–π precision).

F1 –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç Precision –∏ Recall:

[

F1 = 2 \cdot \frac{precision \cdot recall}{precision + recall}

]

–ï—Å–ª–∏ –æ–¥–∏–Ω –∏–∑ –Ω–∏—Ö –±–ª–∏–∑–æ–∫ –∫ –Ω—É–ª—é ‚Üí F1 —Ç–æ–∂–µ –Ω–∏–∑–∫–∏–π.

2.3. Macro-F1 vs Weighted-F1

–í –º–Ω–æ–≥–æ –∫–ª–∞—Å—Å–æ–≤—ã—Ö –∑–∞–¥–∞—á–∞—Ö (bug/feature/billing/other) –º–æ–∂–Ω–æ:

  • —Å—á–∏—Ç–∞—Ç—å F1 –ø–æ –∫–∞–∂–¥–æ–º—É –∫–ª–∞—Å—Å—É, –∞ –ø–æ—Ç–æ–º —É—Å—Ä–µ–¥–Ω—è—Ç—å:
    • macro-F1 ‚Äî —Å—Ä–µ–¥–Ω–µ–µ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–æ–µ –ø–æ –≤—Å–µ–º –∫–ª–∞—Å—Å–∞–º (–∫–∞–∂–¥—ã–π –∫–ª–∞—Å—Å –≤–∞–∂–µ–Ω –æ–¥–∏–Ω–∞–∫–æ–≤–æ),
    • weighted-F1 ‚Äî —Å—Ä–µ–¥–Ω–µ–µ, –≥–¥–µ –∫–∞–∂–¥—ã–π –∫–ª–∞—Å—Å –≤–∑–≤–µ—à–µ–Ω –ø–æ —Å–≤–æ–µ–π –¥–æ–ª–µ –≤ –¥–∞–Ω–Ω—ã—Ö.
  • –ï—Å–ª–∏ –∫–ª–∞—Å—Å—ã –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω—ã, weighted —á–∞—Å—Ç–æ –±–æ–ª–µ–µ ‚Äú—Å—Ç–∞–±–∏–ª—å–Ω—ã–π‚Äù.
  • macro –ø–æ–º–æ–≥–∞–µ—Ç –ø–æ–Ω—è—Ç—å, –Ω–µ –ø—Ä–æ–≤–∞–ª–∏–≤–∞–µ–º –ª–∏ –º—ã —Ä–µ–¥–∫–∏–µ –∫–ª–∞—Å—Å—ã.

–í —ç—Ç–æ–º —É—Ä–æ–∫–µ –±—É–¥–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å weighted F1 –∫–∞–∫ ‚Äú–æ—Å–Ω–æ–≤–Ω—É—é‚Äù –º–µ—Ç—Ä–∏–∫—É.

2.4. –ü–æ—á–µ–º—É –æ–¥–Ω–æ–≥–æ loss –º–∞–ª–æ

Loss ‚Äî —ç—Ç–æ —Å–∏–≥–Ω–∞–ª –¥–ª—è –æ–±—É—á–µ–Ω–∏—è, –Ω–æ:

  • –æ–Ω –Ω–µ –≤—Å–µ–≥–¥–∞ –∏–Ω—Ç—É–∏—Ç–∏–≤–µ–Ω,

  • –æ—Å–æ–±–µ–Ω–Ω–æ –ø—Ä–∏ –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∫–ª–∞—Å—Å–∞—Ö.

3. –¢–µ–æ—Ä–∏—è: compute_metrics –∏ –≤—ã–±–æ—Ä –ª—É—á—à–µ–π –º–æ–¥–µ–ª–∏

3.1. –ö–∞–∫ Trainer –ø–µ—Ä–µ–¥–∞—ë—Ç –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—è –≤ compute_metrics

Trainer –ø—Ä–∏ eval –¥–µ–ª–∞–µ—Ç –ø—Ä–∏–º–µ—Ä–Ω–æ —Å–ª–µ–¥—É—é—â–µ–µ:

  1. –ü—Ä–æ–≥–æ–Ω—è–µ—Ç eval-–¥–∞—Ç–∞—Å–µ—Ç —á–µ—Ä–µ–∑ –º–æ–¥–µ–ª—å ‚Üí —Å–æ–±–∏—Ä–∞–µ—Ç logits –∏ labels.
  1. –§–æ—Ä–º–∏—Ä—É–µ—Ç –æ–±—ä–µ–∫—Ç EvalPrediction, –≤ –∫–æ—Ç–æ—Ä–æ–º:

    • label_ids ‚Äî –∏—Å—Ç–∏–Ω–Ω—ã–µ –º–µ—Ç–∫–∏.
  1. –ü–µ—Ä–µ–¥–∞—ë—Ç —ç—Ç–æ—Ç –æ–±—ä–µ–∫—Ç –≤ compute_metrics.

–ú—ã –º–æ–∂–µ–º –ø–∏—Å–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏—é –∫–∞–∫:

def compute_metrics(eval_pred):

    ...

(—Ç–∞–∫ –∫–∞–∫ EvalPrediction —Ä–∞—Å–ø–∞–∫–æ–≤—ã–≤–∞–µ—Ç—Å—è –≤ (predictions, label_ids)).

–î–∞–ª—å—à–µ:

  • logits —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç—å—é [num_samples, num_labels],
  • labels —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç—å—é [num_samples].

–ù–∞—à–∞ –∑–∞–¥–∞—á–∞:

  • –≤–∑—è—Ç—å argmax –ø–æ –ª–æ–≥–∏—Ç–∞–º ‚Üí –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω—ã–π –∫–ª–∞—Å—Å,
  • –ø–æ—Å—á–∏—Ç–∞—Ç—å –º–µ—Ç—Ä–∏–∫–∏ (accuracy, f1, –∏ —Ç.–ø.),
  • –≤–µ—Ä–Ω—É—Ç—å –∏—Ö –≤ –≤–∏–¥–µ —Å–ª–æ–≤–∞—Ä—è.

Trainer –ø–æ—Ç–æ–º:

  • –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç —ç—Ç–∏ –º–µ—Ç—Ä–∏–∫–∏ –≤ –ª–æ–≥,
  • –º–æ–∂–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ–¥–Ω—É –∏–∑ –Ω–∏—Ö –∫–∞–∫ metric_for_best_model.

  • –ø–æ—Å–ª–µ –æ–±—É—á–µ–Ω–∏—è —É —Ç–µ–±—è –≤ trainer.model –±—É–¥—É—Ç –≤–µ—Å–∞ –ª—É—á—à–µ–≥–æ —á–µ–∫–ø–æ–∏–Ω—Ç–∞.

–°–Ω–∞—á–∞–ª–∞ ‚Äî –ø–æ–ª–Ω—ã–π —Å–∫—Ä–∏–ø—Ç, –ø–æ—Ç–æ–º —Ä–∞–∑–±–æ—Ä.

# train_tickets_metrics.py

import numpy as np
from sklearn.metrics import accuracy_score, precision_recall_fscore_support

from transformers import (
    AutoModelForSequenceClassification,
    TrainingArguments,
    Trainer,
)
from data_pipeline import load_and_prepare_data


def compute_metrics(eval_pred):
    logits, labels = eval_pred

    # 1. –ü—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω—ã–π –∫–ª–∞—Å—Å ‚Äî argmax –ø–æ –ª–æ–≥–∏—Ç–∞–º
    preds = np.argmax(logits, axis=-1)

    # 2. Accuracy
    acc = accuracy_score(labels, preds)

    # 3. Precision, Recall, F1 (weighted)
    precision, recall, f1, _ = precision_recall_fscore_support(
        labels,
        preds,
        average="weighted",
        zero_division=0,  # —á—Ç–æ–±—ã –Ω–µ –ø–∞–¥–∞—Ç—å, –µ—Å–ª–∏ –∫–∞–∫–æ–π-—Ç–æ –∫–ª–∞—Å—Å –Ω–µ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–ª—Å—è
    )

    return {
        "accuracy": acc,
        "f1": f1,
        "precision": precision,
        "recall": recall,
    }


def main():
    csv_path = "tickets.csv"
    base_model_name = "distilbert-base-uncased"

    # 1. –î–∞–Ω–Ω—ã–µ
    (
        tokenized_dataset,
        label2id,
        id2label,
        tokenizer,
        data_collator,
    ) = load_and_prepare_data(
        csv_path=csv_path,
        model_name=base_model_name,
        max_length=128,
        test_size=0.1,
        seed=42,
    )

    # 2. –ú–æ–¥–µ–ª—å
    num_labels = len(label2id)
    model = AutoModelForSequenceClassification.from_pretrained(
        base_model_name,
        num_labels=num_labels,
        id2label=id2label,
        label2id=label2id,
    )

    # 3. –ê—Ä–≥—É–º–µ–Ω—Ç—ã –æ–±—É—á–µ–Ω–∏—è (—Ç–µ–ø–µ—Ä—å —Å –≤—ã–±–æ—Ä–æ–º –ª—É—á—à–µ–π –º–æ–¥–µ–ª–∏)
    training_args = TrainingArguments(
        output_dir="checkpoints/tickets_metrics",
        evaluation_strategy="epoch",
        save_strategy="epoch",
        learning_rate=2e-5,
        per_device_train_batch_size=16,
        per_device_eval_batch_size=32,
        num_train_epochs=3,
        weight_decay=0.01,
        logging_steps=50,
        load_best_model_at_end=True,   # <--- —Ç–µ–ø–µ—Ä—å –≤–∫–ª—é—á–µ–Ω–æ
        metric_for_best_model="f1",    # <--- –ø–æ –∫–∞–∫–æ–π –º–µ—Ç—Ä–∏–∫–µ –≤—ã–±–∏—Ä–∞—Ç—å —á–µ–∫–ø–æ–∏–Ω—Ç
        greater_is_better=True,        # <--- F1: —á–µ–º –±–æ–ª—å—à–µ, —Ç–µ–º –ª—É—á—à–µ
    )

    # 4. Trainer
    trainer = Trainer(
        model=model,
        args=training_args,
        train_dataset=tokenized_dataset["train"],
        eval_dataset=tokenized_dataset["validation"],
        tokenizer=tokenizer,
        data_collator=data_collator,
        compute_metrics=compute_metrics,
    )

    # 5. –û–±—É—á–µ–Ω–∏–µ
    trainer.train()

5.1. compute_metrics

def compute_metrics(eval_pred):
    logits, labels = eval_pred

    preds = np.argmax(logits, axis=-1)

    acc = accuracy_score(labels, preds)

    precision, recall, f1, _ = precision_recall_fscore_support(
        labels,
        preds,
        average="weighted",
        zero_division=0,
    )

    return {
        "accuracy": acc,
        "f1": f1,
        "precision": precision,
        "recall": recall,
    }
